How it works

Type a sentence and step through exactly what ELIZA does with it.

Responses by Anthony Hay’s CC0 ELIZA engine - the same engine you can talk to on the Try ELIZA page.

This visualisation repurposes Anthony Hay’s faithful reconstruction of ELIZA (released CC0 public domain): his engine generates the responses exactly as Weizenbaum’s 1966 program would, and we use it to demo each step.

His code runs essentially unchanged. We lifted the engine out of his eliza.html (removing only its on-screen terminal) and left the algorithm untouched, then wrapped it in a small harness that loads the 1966 DOCTOR script and runs a single response. It talks to the demo through Hay’s own tracer interface: ELIZA already supports a step-by-step trace mode, so we add a tracer that records each step the engine takes (keyword ranking, decomposition, reassembly, memory), and the display turns those records into the panels above. Both the harness and tracer (eliza-hay.js) and the display (trace.js) are open.
